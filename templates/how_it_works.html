How It works

✨ Magic of course ✨
By magic I mean we use the github api and some backend processing of the data returned.

Okay cool story. How do I use it?

First step

Add your boiler plate code and then add your tutorial intro to your README.md
Make sure to add linebreaks at the end of your file! If not, github see's the new line breaks in your next commit as a changed line and it adds it to the next step.

Next Step

Commit all your changes, and then move on to your next step. You will just add the next step's code changes and then add your instructions into the Readme.md like before!

Rinse And Repeat.
